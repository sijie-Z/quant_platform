# Trading Spec: 80-Day Reversal Strategy

> **状态**：v1 production spec
> **验证**：in-sample Sharpe 0.45 (裸) / 0.60 (vol filter)，OOS 待验证
> **Base**：RQ5b + Ceiling Probe + Regime Discovery v4

---

## 1. 策略定义

### 1.1 Entry Rule

| 参数 | 值 | 来源 |
|------|-----|------|
| 信号窗口 | 40 个交易日 | Ceiling Probe (S=40 为尖锐峰) |
| 持有期 | 80 个交易日 | Ceiling Probe (H=80 为尖锐峰) |
| 选股范围 | 全样本 ~101 只 A 股 | — |
| 选股比例 | 20%（跌幅最大的) | RQ5b |
| 权重 | 等权 | Discovery v3 协议 |
| Rebalance | 每 80 个交易日 (约一季) | — |

**Entry 逻辑**：
```
每个调仓日:
  1. 计算所有股票过去 40 日累计收益 (compound return)
  2. 选择收益最低的 20%（跌幅最大的）
  3. 等权配置 (1/selected_count)
  4. 持有 80 个交易日
  5. 80 日后的调仓日一次性卖出，同时开新仓
```

### 1.2 Vol Filter

| 参数 | 值 | 来源 |
|------|-----|------|
| 波动率指标 | 等权市场日收益的 20 日滚动标准差 | Ceiling Probe |
| 阈值 | 70% 分位数 (只保留低/中波动期) | Ceiling Probe |
| 触发动作 | 跳过该期交易 (不持仓, 持有现金) | — |

**Filter 逻辑**：
```
在调仓日，如果 market_vol_20d > 历史 70% 分位数:
  跳过本次调仓 (不买入, 不持仓)
否则:
  正常执行 Entry 逻辑
```

### 1.3 Exit Rule

| 条件 | 动作 |
|------|------|
| 持仓满 80 个交易日 | 一次性卖出 (在下次调仓日) |
| Vol filter 触发时 | 不持仓 (现金) |
| 被动 exit | 无提前止损/止盈 |

### 1.4 Position Sizing

| 参数 | 值 |
|------|-----|
| 单票权重 | 1/N (N = 选中的股票数, ~20) |
| 最大单票 | 5% (天然满足, ~20 只 × 5% = 100%) |
| 现金权重 | Vol filter 触发时 100% 现金 |
| 杠杆 | 无 (纯多头) |

## 3. OOS 验证计划

### 3.1 时间分割

| 指标 | 最低接受 | 目标 |

## 4. 已知风险

| 风险 | 影响 | 应对 |
|------|------|------|
| Vol filter 减少交易次数 | 样本量下降, Sharpe 估计方差增大 | 接受, 不修改 |
| 80d 持有期内状态切换 | 持仓可能跨 Bull→Bear 转换 | 固有风险, 不应对 |
| 101 只样本 | 代表性问题 | 扩展到 CSI300 后验证 |
| Transaction cost 未计入 | 实盘 Sharpe 可能低于 in-sample | OOS 后加成本跑 |
