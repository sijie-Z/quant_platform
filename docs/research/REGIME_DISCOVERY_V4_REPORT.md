# Regime Discovery v4 — 市场状态条件化结构报告

> **日期**：2026-06-21
> **核心问题**：α ≠ 静态信号。α = f(市场状态)。反转结构在什么状态下有效？

---

## 执行摘要

**反转策略只在方向性市场（上涨/下跌）中有效，在震荡/高波市场中失效。**

这解释了 RQ5b 发现的 "Sharpe 0.45 但滚动 52% 正收益" 的矛盾——全周期结果掩盖了完全不同的状态依赖表现。

3. **这不是因子选择问题，是状态依赖问题**——改变因子不能解决，只有改变策略的条件才能解决。

## 状态持续性分析

| 状态 | 平均持续期 | 最长持续期 | 出现次数 |
|------|-----------|-----------|---------|
| Bull | ~45 天 | ~200 天 | ~15 次 |
| Bear | ~50 天 | ~160 天 | ~18 次 |
| Sideways | ~55 天 | ~220 天 | ~18 次 |

**状态平均持续 45-55 天（约 2-3 个月）。** 这意味着：
- 反转策略的最优持有期（80 天）与平均状态持续期相当
- 一个持仓周期内很可能经历状态切换
- 纯被动反转策略会在状态切换点承受损失

## 综合结论

### 最终市场结构模型

```
α = Reversal(过去收益) × I(市场有方向) + Noise × I(市场无方向)

其中: 
  I(市场有方向) = Bull ∪ Bear ∪ LowVol
  I(市场无方向) = Sideways ∪ HighVol
```

### 三大核心发现

1. **A 股反转结构确认存在**，最优参数 S=40/H=80，全周期 Sharpe=0.45。
2. **但该结构完全由方向性市场驱动**。在 Sideways 市场中，反转策略 Sharpe 为负。
3. **全周期 Sharpe=0.45 是 Bull(+1.38) + Bear(+0.70) + Sideways(−0.46) 的加权平均**。不是"稳定的 0.45"，而是"约 2/3 时间有效、1/3 时间亏钱的混合体"。

### 研究意义

```
从:   α = Signal
到:   α = Signal × I(Market has direction)
```

这意味着：
- 纯静态反转策略不是最优解
- 需要状态感知机制——在 Sideways/HighVol 时减仓或切换策略
- 最优持有期（80d）与平均状态持续期（45-55d）的差异是天然的风险来源
