## 执行摘要

**A 股（2018-2025）在所有测量时间尺度上均呈现均值回归。不存在任何使趋势延续策略有效的时间窗口。**

IC(Horizon) 曲线在 5 日到 240 日的全部 10 个测量点上均为负值。最强反转发生在 80 日窗口（IC=-0.081，t=-4.58，统计学显著）。反转效应在小盘股中略强于大盘股。行业样本量不足，无法得出可靠结论。

### 关键发现

1. **IC 在全部 10 个 horizon 上为负**。零个例外。A 股在 2018-2025 年间**没有任何时间尺度支持趋势延续**。

2. **最强反转在 80 日**。IC=-0.0808，t-stat=-4.58（p < 0.0001）。这是统计学上极显著的结果。"过去 80 日涨得越多，未来 80 日跌得越多"是 A 股最可靠的预测模式。

3. **IC 曲线呈 U 形**：短端（5-10 日）反转较弱（IC≈-0.035），中段（20-120 日）反转最强（IC≈-0.06 至 -0.08），长端（160-200 日）反转趋于零（IC≈-0.01 至 -0.03）。

4. **200 日窗口最接近随机游走**。IC=-0.01，t=-0.58，无法拒绝 H₀（IC=0）。这是唯一"真正无预测力"的窗口——动量学派最常用的 200 日均线在此区间失效。

5. **240 日窗口 IC 回升至 -0.04**。这与 200 日的不连续性暗示可能存在年频日历效应（如春节效应、财报季节）。需要进一步验证。

6. **pos_ratio 在 31-51% 之间**。这意味着即使 IC 均值为负，仍有约 30-40% 的时间过去-未来收益同方向。纯反转策略在这些时间会亏损。这解释了为什么简单的反转信号难以获利（RQ5a 回测全负）。

## RQ6: 市值分层

| Size | IC_5d | IC_20d | IC_40d | IC_60d | IC_120d | IC_240d |
|------|-------|--------|--------|--------|---------|---------|
| **Small** | -0.039 | **-0.094** | **-0.081** | **-0.098** | **-0.100** | +0.004 |
| **Mid** | -0.028 | -0.058 | -0.052 | -0.032 | -0.063 | -0.088 |
| **Large** | -0.045 | -0.049 | -0.057 | -0.064 | -0.046 | -0.056 |

### 关键发现

1. **反转效应在三个市值组中都存在**。不是小盘独有的现象。

2. **小盘股的反转强度最大**。IC_60d=-0.098，IC_120d=-0.100——小盘股过去半年的涨跌对未来半年有最强的预测力。

3. **中大盘股的反转强度接近**。Mid 和 Large 的 IC(H) 曲线形状相似，幅度略低于 Small。

4. **小盘股 240 日 IC≈0**。唯一接近零的测量点——小盘股的年度反转不显著，这可能与 A 股小盘股的年度轮动效应有关。

5. **市值差异在 60-120 日最大**。中型股 vs 小型股在中期窗口的 IC 差异约 0.04-0.05，不算大。

**判定**：H₀ 未被拒绝——反转效应存在，但市值分层的分化不足以解释整体现象。反转是A股整体性质，不是小盘独有的。

### 关键发现

1. **数据限制**：仅 4 个行业有 ≥5 只股票，样本量严重不足（101 只分散到 30+ 行业）。

2. **F51（金融/银行）反转最强**：IC_120d=-0.163，显著高于市场平均值。金融股呈现强均值回归——这可能与 A 股银行股的"国家队"调控效应有关。

3. **K70（公用事业）反转最弱**：IC 接近零。公用事业股在 A 股可能更接近随机游走。

4. **"Unknown"行业异常**：IC 在 5d、60d、120d 为正——这是唯一出现正 IC 的行业。但这个组是"未分类"，可能混合了不同类型的股票，不可信。

**判定**：样本量不足，无法得出可靠结论。需要更大数据集（500+ 股票）来分解行业效应。

## 综合结论

### A 股市场结构画像（2018-2025）

```
                   均值回归强度
                    │
      ┌─────────────┼─────────────┐
      │ 80d 最强    │             │
      │ IC=-0.081   │             │
      │ t=-4.58     │             │
      │             │             │
短周期←─5d────20d────80d───160d──240d──→长周期
      │             │    │       │
      │ 全区间负IC  │   200d≈0  │
      │ 无趋势窗口  │   随机游走 │
      │             │           │
      └─────────────┴───────────┘
```

1. **A 股不是"没有 Alpha"，是 Alpha 的符号与美国市场相反**。过去涨 → 未来跌（均值回归），而非过去涨 → 未来涨（动量）。

2. **不同时间尺度的反转强度不同**。80 日最强（年化信息比最高），5-10 日最短/最弱，160-200 日趋于零。这意味着反转交易需要选择最优时间窗口。

3. **反向交易（Anti-Momentum）本身不产生正收益**（RQ5a 全部负 Sharpe）。原因：虽然方向正确，但月度调仓频率与最优反转窗口（5-80 日）不匹配，且 30-40% 的时间同方向运动造成亏损。

4. **纯反转策略不足以产生稳定 Alpha**。需要更精确的 Entry/Timing——识别反转发生的具体时机，而非简单每月做空过去赢家。

### 对后续研究的含义

| 发现 | 研究含义 |
|------|---------|
| IC(H) 全为负 | 任何趋势策略在 A 股期望收益为负 |
| 80 日反转最强 | 最优信号窗口 ≈ 80 日 |
| pos_ratio ≈ 30-40% | 反转不是 100% 可靠，需要择时 |
| 小盘反转更强 | 信号可叠加市值因子 |
| 月频调仓不匹配 RQ5a | 需要更高频调仓（周/双周） |
| 反转不足以盈利 | 需要复合信号（反转 + 市值 + 行业 + 状态） |

### 方法论的自我审视

**强反转（IC 全负）而回测全亏（Sharpe 全负）意味着什么？**

这个看似矛盾的结果指向一个关键认识：**IC 衡量的是二元方向性，而非可交易性**。IC 为负确认方向正确，但实际交易中：
- 月频调仓滞后于反转的最优时效窗口
- 等权优化器在高 IC 时期未能充分暴露
- 交易成本侵蚀了边缘收益

**结论：方向对了只是第一步。信号提取、执行频率、组合构建三者缺一不可。**
