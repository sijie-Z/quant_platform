# Market Structure Discovery v3 — 实验协议

> **冻结日期**：2026-06-21
> **状态**：已冻结
> **前序研究**：RQ4 确认趋势因子在 A 股系统性方向错误（4/5 簇 8/8 年 IC 为负）
> **核心问题**：A 股的反转到底发生在哪个时间尺度？是否依赖于市值和行业？

---

## 1. 研究背景

RQ4 建立了以下事实：
- 传统趋势因子（动量 1m/3m/6m/12m、RSI、换手率）在 A 股全部方向错误
- 4/5 簇在全部 8 年和全部 3 个市场状态中 IC 一致为负
- 这不是"没有信号"，而是**信号极性反转**——A 股呈现均值回归而非趋势延续

但时间维度仍然模糊：
- ShortReversal 簇包含 1m 动量 + RSI(14) + BreakoutProximity
- MediumTrend 包含 3m/6m 动量 + TrendStage
- 这些簇的因子混合了多个时间尺度
- **无法精确知道反转发生在几日的窗口**

本协议设计三个研究问题，系统测绘 A 股的可预测性结构。

## 3. 研究问题与实验设计

---

### 研究问题 5（RQ5）：A 股反转的时间尺度

**问题**：反转信号在哪个持有期最强？

**假设**：
- **H₀**：反转效应在所有时间尺度上一致
- **H₁**：反转存在最优时间尺度（如 10-20 日最强，长周期消失或翻转为趋势）

#### 实验 5.1 — IC(Horizon) 曲线

对每个 horizon H ∈ {5, 10, 20, 40, 60, 80, 120, 240}：

1. 计算每个股票每日的 **过去 H 日累计收益** 作为因子值
2. 计算该因子与 **未来 H 日累计收益** 的 Spearman Rank IC
3. 每 20 个交易日采样一次（避免重叠窗口伪像）
4. 记录：`mean_ic`, `icir`, `t_stat`, `pos_ratio`

输出：IC(H) 曲线——H 为横轴，IC 为纵轴。

#### 实验 5.2 — 反转 Signal 回测

对每个 horizon H，构建反转信号：
- Signal = −rank(past_H_return)  （反转信号：过去涨得多 → 负权重）
- 如果 H₀ 成立（反转一致），每个 H 的 Sharpe > 0
- 如果 H₁ 成立，只在某些 H 上 Sharpe > 0

回测参数使用 Discovery v2 的相同标准参数（月频调仓、等权优化器）。

#### 判定规则

如果 IC 曲线在短端（H ≤ 40）为负且在长端（H ≥ 120）翻转为正或变弱：**接受 H₁，反转是短周期现象**。

---

### 研究问题 6（RQ6）：市值分层

**问题**：反转效应在大/中/小盘股中是否一致？

**假设**：
- **H₀**：反转效应与市值无关
- **H₁**：反转效应主要存在于小盘股（大盘股偏趋势/随机游走）

#### 实验 6 — 市值分层分析

1. 每月按市值将股票等分为 Small/Mid/Large 三组
2. 对每组分别运行 RQ5（全 H 扫描）
3. 每组独立输出 IC(H) 曲线

关注点：
- Small 组的 IC(H) 与 Large 组是否有显著差异？
- 是否存在"小盘反转、大盘趋势"的结构？

#### 判定规则

如果两个极端组的 IC(H) 曲线在 ≥3 个 H 点上的差异超过 0.02（IC 绝对值）：**接受 H₁，市值显著影响反转结构**。

---

### 研究问题 7（RQ7）：行业差异

**问题**：反转效应是 A 股整体现象，还是由特定行业驱动？

**假设**：
- **H₀**：反转效应在行业间一致
- **H₁**：部分行业呈反转，部分行业呈趋势延续

#### 实验 7 — 行业分解

1. 按一级行业分类（~28 个申万行业）
2. 对每个有 ≥5 只股票的行业，运行 IC(H) 扫描（H ∈ {5, 20, 60, 120}）
3. 输出每个行业的短/中/长期 IC

#### 判定规则

如果有 ≥3 个行业的 IC 符号与整体不一致（如整体为负 IC，某行业为正 IC）：**接受 H₁，行业间存在差异**。

## 4. 评价指标

| 指标 | 说明 |
|------|------|
| IC(H) | 在 horizon H 上的 Spearman Rank IC 均值 |
| ICIR(H) | IC 均值 / IC 标准差 |
| t_stat | IC 是否显著异于 0 |
| pos_ratio | IC 为正的天数占比 |
| Sharpe | 月频回测的年化 Sharpe Ratio |
